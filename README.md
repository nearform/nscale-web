
# nscale-web

The Web GUI of [nscale](http://github.com/nearform/nscale).

Install
------

```sh
git clone git@github.com:nearform/nscale-api.git
cd nscale-web
npm install
npm install -g bower
bower install
```

Usage
-----

```sh
node server
```

And open [localhost:9000](http://localhost:9000)
